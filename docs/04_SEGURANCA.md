# Seguranca e cuidado

O projeto deve proteger a rotina da usuaria com regras simples.

## Cuidados do produto

- Confirmar toda venda antes de salvar.
- Permitir desfazer a ultima acao nas proximas fases.
- Evitar telas cheias.
- Manter historico das alteracoes importantes.
- Evitar depender de login de terceiros no MVP.

## Dados pessoais

Guardar apenas o necessario para controle familiar: nome do cliente, valor, produto e status do pagamento.
