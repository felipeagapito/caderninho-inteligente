# Visao geral

O Caderninho Inteligente e uma ferramenta familiar, independente e mobile-first para ajudar uma vendedora por catalogo a registrar vendas, clientes, estoque e valores em aberto.

## Publico principal

Uma pessoa que ja usa caderninho fisico e precisa de uma transicao simples para o digital.

## O que o produto deve parecer

- Um app de celular, nao um sistema corporativo.
- Letras maiores e telas limpas.
- Poucas escolhas por tela.
- Confirmacao antes de salvar.
- Resumo claro do que vendeu, do que falta receber e do que precisa repor.

## O que fica fora do MVP

- Integracao com API de marca.
- Automacao de WhatsApp.
- APK nativo.
- Audio gravado dentro do app.
- Banco real.
