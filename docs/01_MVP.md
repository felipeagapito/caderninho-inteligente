# MVP

## Objetivo

Entregar uma primeira versao usavel no celular, com visual limpo e dados mockados, para validar a experiencia antes de banco, IA real e audio.

## Escopo da primeira versao

1. Tela inicial com resumo.
2. Tela de vendas com campo grande para ditado pelo teclado do celular.
3. Tela de estoque com produtos e alerta de reposicao.
4. Tela de clientes com valores em aberto.
5. Tela de resumo.
6. PWA instalavel por link.

## Regra central

Nenhuma venda deve ser salva sem uma etapa de confirmacao clara.
